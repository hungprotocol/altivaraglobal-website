# SEO Plan

Working notes. Refine as we learn what queries actually convert.

## Target queries (initial hypotheses)

Primary:

- "business operations consulting Singapore"
- "operating model design Philippines"
- "managed workforce solutions Southeast Asia"
- "enterprise delivery consulting APAC"

Secondary:

- "AI enablement consulting Singapore"
- "governance and performance management consulting"
- "operations execution consulting Asia-Pacific"

## On-page checklist (per page)

- [ ] Unique `<title>` under 60 characters
- [ ] Unique `<meta name="description">` under 160 characters
- [ ] One `<h1>` per page, matching the search intent
- [ ] Open Graph tags (`og:title`, `og:description`, `og:image`, `og:url`)
- [ ] Twitter card tags
- [ ] Canonical URL
- [ ] `lang="en"` on `<html>` (already set)

## Structural

- [ ] `sitemap.xml` listing all five pages
- [ ] `robots.txt` allowing crawl, pointing to sitemap
- [ ] Schema.org `Organization` JSON-LD on `index.html`
- [ ] Schema.org `LocalBusiness` if a physical office address is published
- [ ] Submit sitemap to Google Search Console and Bing Webmaster

## Off-page (slow burn)

- ACRA / corporate registry listings reference the website
- LinkedIn company page links to the site
- Industry directory submissions (consulting directories, APAC business
  associations) where appropriate
- Earned media via thought-leadership content (timing TBD)

## Measurement

- Search Console: indexed pages, impressions, top queries, CTR
- Lightweight server-side analytics if/when added (avoid Google
  Analytics in early phase — keep privacy posture clean for the legal
  pages we already publish)
